# 법정동코드표 (lawd-codes)

> `items.lawd_cd`(SPEC-COLLECTOR-001 3계층 매핑 결과)와 국토부 실거래가 조회
> (`LAWD_CD` 파라미터, SPEC-ENRICH-001)에 쓰이는 법정동코드(시군구 5자리) 참고표입니다.

## 코드 형식

- 형식: 법정동코드 앞 5자리(시군구 단위). 예: `28260`(인천 서구).
- MOLIT `getRTMSDataSvcAptTradeDev` 의 `LAWD_CD` 파라미터로 그대로 사용됩니다.

## 동봉 파일

- `docs/lawd-codes.csv` — **워치리스트 관심 지역 위주의 소규모 샘플 subset** 입니다.
  전체 시군구를 담지 않습니다.

## 출처(Provenance)

- 소스: 행정표준코드관리시스템(www.code.go.kr) "법정동코드 전체자료".
- 스냅샷 일자: 2026-07-03 기준 수기 발췌(샘플 10건).
- 행정구역 개편(통합·분리·신설)이 있으면 코드가 변동될 수 있습니다.

## 백로그 (good-first-issue)

- 전체 법정동코드 CSV 동봉 + 자동 생성 스크립트(행정표준코드 원본 → 5자리 시군구 축약)
  작성은 good-first-issue 대상입니다. (기획서 §6.4·§13)
- 행정구역 개편 대응(스냅샷 일자 갱신·차분 검토)도 함께 다룹니다.
