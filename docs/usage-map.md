# 용도 매핑표 (usage-map)

> 원문 용도(courtauction.go.kr `usage`)를 표준 카테고리로 접는 규칙입니다.
> 소스: `packages/core/src/usage-map.ts` (단일 진실 공급원). 이 문서는 그 표를 공개용으로 옮긴 것입니다.

## 표준 카테고리

`아파트` · `빌라` · `오피스텔` · `상가` · `토지` · `기타`

## 매핑 규칙

| 원문 용도 | 표준 카테고리 |
|---|---|
| 아파트, 주상복합 | 아파트 |
| 연립주택, 다세대주택, 연립다세대, 빌라, 다가구주택 | 빌라 |
| 오피스텔 | 오피스텔 |
| 근린생활시설, 근린상가, 상가, 점포, 상업용, 사무실 | 상가 |
| 토지, 대지, 임야, 전, 답, 과수원, 잡종지, 농지 | 토지 |
| (그 외 미매핑) | 기타 + 경고 로그 |

## 미매핑 처리 (REQ-019)

- 표에 없는 원문 용도는 표준 카테고리 **기타**로 분류하고 경고 로그를 남깁니다.
- 미매핑 사례는 good-first-issue 로 수집해 `usage-map.ts` 표에 추가합니다.

## 기여 방법

1. 실제 공고에서 발견한 미매핑 용도 문자열을 이슈로 등록
2. `packages/core/src/usage-map.ts` 의 `USAGE_MAP` 에 항목 추가
3. `packages/core/tests/usageMap.test.ts` 에 검증 케이스 추가
4. 본 문서 표 갱신
