# 아파트 할인율 수동 대조 20건 (enrich-validation)

> Traceability: SPEC-ENRICH-001 · 기획서 §12 (M3 DoD)
>
> 산출 할인율(`discountRate`)과 **수동 계산** 결과를 대조하는 시트입니다.
> 할인율 = `1 − (최저매각가 / 실거래 중위값)`, 통계는 **중위값(median)** 사용(평균 금지).
> 불일치 건은 원인(정규화 실패·면적 밴드·표본 편향 등)을 분석해 비고에 기록합니다.

## 계산 규칙 요약

- 중위값: 표본 정렬 후 홀수면 정중앙, 짝수면 중앙 2값 평균.
- 표시 할인율(%) = `−round(discountRate × 100)`.
- 신뢰도: 동일 단지 ≥5 → 높음 / 동일 3~4 또는 폴백 ≥10 → 보통 / 그 외 → 낮음("참고치 (표본 부족)").
- 빌라·토지 → 표본 무관 "참고치" 고정, 강조 억제.
